# Tweet Writer

チャトレ/ライバー事務所向けバズるツイートを作成するスキル。

## 参照データ

**自社ツイート実績：**
- `chatre1_tweets.json` - @mic_chat_ チャトレ事務所（19件）
- `chatre2_tweets.json` - @ms_stripchat チャトレ事務所（19件）
- `liver_tweets.json` - @tt_liver ライバー事務所（20件）

**成功パターン分析：**
- `success_patterns.json` - 高スコア投稿パターン（37件、平均スコア7.7）

**他人のバズ投稿（参考用）：**
- `trending_posts.json` - 業界問わずバズ投稿（20件以上、エンゲージ最大22,472）

## 使い方

`/tweet-writer [テーマ]` で呼び出し。

## ターゲット層

**チャトレ向け：**
- 主婦・ママ（スキマ時間、通勤ゼロ）
- 30代・40代（年齢を活かす、顔出しなし）
- 夜職経験者（非接触、高収入）
- 副業希望者（本業バレ防止）
- 完全未経験（サポート充実）

**ライバー向け：**
- 副業希望（本業＋配信）
- 顔出しなし希望
- 在宅ワーク希望

## 成功パターン

1. **Q&A形式**: 「〇〇って思ってない？」→ 解決策
2. **共感→解決**: 悩みに共感 → うちなら解決
3. **実績→方法**: 具体的数字 → やり方
4. **ストーリー**: 所属の子の実体験
5. **リスト形式**: 箇条書きで価値提供
6. **逆張り**: 「〇〇は嘘」「〇〇は幻想」

## バズ投稿の特徴（trending_posts.jsonより）

- 具体的な数字（3年で年収300万、月3万、7時間睡眠）
- リスト形式（3〜5個の箇条書き）
- 「ぶっちゃけ」「正直」「マジで」フック
- 常識否定 → 本質提示
- 過去の自分への手紙形式

## ベネフィット例

- 高収入（時給5000円〜、月30万〜50万）
- 顔出しなし / 非接触 / 完全在宅
- 日払い対応 / 時間自由 / 通勤ゼロ
- 年齢不問 / サポート充実

## フォーマット

- 200-270文字
- ハッシュタグ禁止
- 絵文字1-2個
- 最後にCTA（DMで〇〇と送って）

## 注意点

- 外部リンクは避ける（ペナルティ）
- 80%価値提供、20%宣伝
- 投稿後1時間が勝負
