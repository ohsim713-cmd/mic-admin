# 📋 並列開発・各員個別指示書 (Instance Directives)

このドキュメントは、4つの Claude Code インスタンスそれぞれが最初に読み込むための指示書である。

---

## 🛰️ 1号機：【脳：Orchestrator】への指令
あなたはシステムの核心である「自律会議ロジック」を担当する。
1. `frontend/lib/agents/graph.ts` を作成し、`@langchain/langgraph` でノード（役員）間の循環構造を作れ。
2. `COO（検閲官）` が生成物を採点し、リテイクを命じるループを最優先で実装せよ。
3. `BoardroomState` を定義し、全役員が同じ議事録を見れるようにせよ。

## 🔍 2号機：【目と耳：Research】への指令
あなたは外の世界と結果を知るための「感覚機能」を担当する。
1. `frontend/lib/tools/search.ts` を作成し、GeminiにGoogle検索結果を注入する仕組みを構築せよ。
2. X/InstagramのAPIから最新の投稿数値（いいね、インプ数等）を取得し、`Analyst（分析官）` が読み取れる形式に変換せよ。
3. 定期的にトレンド情報を収集するバックグラウンドタスクを設計せよ。

## 🖼️ 3号機：【顔：UI/UX】への指令
あなたはユーザーが驚く「プレミアムな司令塔画面」を担当する。
1. `frontend/app/dashboard/page.tsx` を作成し、Manus AI のような「自律稼働ログ」を表示せよ。
2. エージェントたちの会議（思考プロセスの断片）を流すアニメーションコンポーネントを作れ。
3. ユーザーが「承認」や「方針修正」をワンタップでできるUIを実装せよ。

## 🎬 4号機：【手足：Production】への指令
あなたはこのシステムの「全業種対応」と「メディア品質」を担当する。
1. `frontend/knowledge/industries/` に、業種ごとの専門知識を注入できるプラグイン構造を作れ。
2. `Google Cloud TTS` を導入し、ショート動画を「高級な声」に差し替えろ。
3. Imagen 3 のプロンプトを極限まで磨き、AIキャラクターが「中の人」に見えるビジュアルを追求せよ。

---

### 全員への共通指示
- コードを書く前に、必ず `frontend/DOCS/AGENT_ARCHITECTURE.md` を読め。
- 自分の作業が他の担当範囲（例：UI担当がAPIを勝手に変える等）を壊しそうな場合は、共通ドキュメントに警告を残せ。
- ライブラリの追加が必要な場合は、`package.json` を更新し、ターミナルで実行コマンドを提案せよ。
