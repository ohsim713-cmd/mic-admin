# /status - 全体ステータス確認スキル

システム全体の状況を一覧表示します。

## 使い方

```
/status
```

## 実行手順

1. **並列で情報収集**

   ```bash
   # ストック状況
   curl -s "http://localhost:3000/api/automation/stock?view=full"

   # 今日のスケジュール
   curl -s "http://localhost:3000/api/automation/post"

   # 統計情報
   curl -s "http://localhost:3000/api/db/stats"
   ```

2. **ダッシュボード形式で報告**

   ```
   📊 システムステータス
   ═══════════════════════════════════

   【ストック状況】
   liver:    ██████████░░░░░ 15/30
   chatre1:  ████░░░░░░░░░░░  8/30  ⚠️ 要補充
   chatre2:  ██████████████░ 28/30

   【今日の投稿】
   完了: 8/15 (53%)
   次回: 14:00 (あと32分)

   【週間パフォーマンス】
   投稿数: 98
   平均スコア: 8.2
   成功パターン: 23件
   ```

## 報告項目

| カテゴリ | 内容 |
|----------|------|
| ストック | 各アカウントの在庫数と補充要否 |
| スケジュール | 今日の投稿進捗と次回投稿時刻 |
| パフォーマンス | 週間の投稿数と平均スコア |
